# 📌 Project Overview

This **Sales CSV Agent Project** focuses on analyzing **sales transaction data** using **AI-powered natural language queries** to uncover meaningful business insights.

Using **Python**, **LangChain**, **Groq LLM**, and **Pandas**, this project transforms raw CSV sales data into actionable insights through **cleaning, analysis, and visualization**.

---

# 🎯 Project Objectives

✅ Load and analyze CSV datasets  
✅ Perform automated data cleaning  
✅ Ask questions in natural language  
✅ Generate intelligent business insights  
✅ Create visual charts and reports  
✅ Detect trends and patterns in sales data

# 🔄 Project Workflow

```text
CSV Dataset 📁
      ↓
Load Dataset using Pandas 🐼
      ↓
Create AI Agent with LangChain 🔗
      ↓
Connect Groq LLM ⚡
      ↓
Ask Questions 💬
      ↓
Generate Insights 📊
```

---

# 💡 Example Queries

```python
agent.invoke("How many rows and columns are in dataset?")
agent.invoke("Describe all columns")
agent.invoke("Show top-selling products")
agent.invoke("Create sales chart")
agent.invoke("Which product is most demanded?")
```
